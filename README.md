# 🫀 Heart Disease Prediction

## 📌 Project Description
This project predicts whether a person has heart disease using machine learning. It uses Logistic Regression and includes data cleaning, EDA, feature scaling, and model evaluation.

---

## 📊 Dataset
- Heart Disease UCI Dataset (Kaggle)
- Target: 0 = No disease, 1 = Disease

---

## ⚙️ Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Scikit-learn

---

## 🧠 Machine Learning Model
- Logistic Regression

---

## 📈 Steps Performed
1. Data Loading
2. Data Cleaning (Missing values handling)
3. Exploratory Data Analysis (EDA)
4. Feature Scaling
5. Train/Test Split
6. Model Training
7. Model Evaluation

---

## 📊 Evaluation Metrics
- Accuracy Score
- Confusion Matrix
- Classification Report
- ROC-AUC Curve

---

## ⭐ Key Features Affecting Prediction
- Age
- Chest Pain Type
- Cholesterol
- Maximum Heart Rate
- Exercise Induced Angina

---

## 🚀 How to Run
1. Upload dataset (heart.csv)
2. Run Jupyter Notebook
3. Execute all cells step by step
